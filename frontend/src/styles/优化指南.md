# 🎨 颜色和阴影优化指南

## 1. 新的颜色系统

### 主色调
- `--primary-color`: #667eea (现代蓝紫色)
- `--primary-light`: #764ba2 (深紫色)
- `--primary-gradient`: 蓝紫色渐变背景

### 辅助色渐变
- `--success-gradient`: 蓝绿色渐变
- `--warning-gradient`: 粉红色渐变  
- `--danger-gradient`: 粉橙渐变
- `--info-gradient`: 蓝粉渐变

### 中性色层次
- 4级文字颜色：primary → secondary → tertiary → muted
- 3级背景色：bg-color → bg-white → bg-gray
- 4级边框色：base → light → lighter → extra-light

## 2. 阴影系统

### 6级阴影层次
```css
.shadow-xs      /* 超小阴影 */
.shadow-sm      /* 小阴影 */
.shadow-base    /* 基础阴影 */
.shadow-md      /* 中等阴影 */
.shadow-lg      /* 大阴影 */
.shadow-xl      /* 超大阴影 */
```

### 特殊阴影效果
- `--shadow-card`: 卡片专用阴影
- `--shadow-button`: 按钮悬停阴影
- `--shadow-hover`: 悬停增强阴影

## 3. 使用方式

### 在Vue组件中使用
```vue
<template>
  <div class="modern-card gradient-bg-primary">
    <h3 class="text-primary">标题</h3>
    <p class="text-secondary">内容</p>
  </div>
</template>

<style scoped>
.custom-card {
  background: var(--bg-white);
  border-radius: var(--border-radius-md);
  box-shadow: var(--shadow-card);
  border: 1px solid var(--border-light);
}
</style>
```

### 在SCSS中使用混合宏
```scss
.my-component {
  @include modern-card;
  @include gradient-bg(success);
  
  &:hover {
    @include shadow-level(lg);
  }
}
```

### 常用CSS类
```html
<!-- 背景色 -->
<div class="bg-white bg-gray bg-primary">

<!-- 文字颜色 -->
<span class="text-primary text-secondary text-muted">

<!-- 圆角 -->
<div class="rounded rounded-md rounded-lg rounded-full">

<!-- 阴影 -->
<div class="shadow-base shadow-lg shadow-xl">

<!-- 渐变背景 -->
<div class="gradient-bg-primary gradient-bg-success">
```

## 4. Element Plus组件优化

### 卡片组件
- 圆角: 8px → var(--border-radius-md)
- 阴影: 基础阴影 → 卡片专用阴影
- 悬停: 添加边框颜色变化和阴影增强

### 按钮组件
- 主要按钮: 应用渐变背景
- 所有按钮: 添加悬停动画效果
- 圆角: 统一为6px

### 表单组件
- 输入框: 聚焦时有发光效果
- 表格: 行悬停有背景色变化
- 标签页: 激活状态有渐变指示器

## 5. 响应式设计

新的间距系统支持更多断点：
- xs: 8px
- sm: 12px  
- base: 16px
- md: 20px
- lg: 24px
- xl: 32px
- 2xl: 40px
- 3xl: 48px

## 6. 暗色主题支持

暗色主题已集成，系统会根据用户偏好自动切换。

## 7. 最佳实践

1. **优先使用CSS变量**：保持样式一致性
2. **合理使用阴影层次**：根据元素重要性选择阴影级别
3. **渐变背景适度使用**：主要用于重要按钮和头部区域
4. **保持对比度**：确保文字在背景上的可读性
5. **利用动效**：适当添加过渡效果提升用户体验
